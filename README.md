# Blakkbrodie
## Introduction
Blakkbrodie is a SA Art Dump website built with Bootstrap and Hugo. It features a simple and responsive design, with a navbar and a collapse section for about and contact information.
## Key Features
- Responsive design
- Navbar with collapse section
- About and contact information
## Tech Stack
- Bootstrap
- Hugo
## Installation
To install the project, clone the repository and run `hugo server` to start the development server.
## Usage
To use the project, simply navigate to the website in your browser. The navbar and collapse section can be used to access the about and contact information.
## Environment Variables
No environment variables are required to run the project.
